## Decisions Made

- **Redirect through API for cookie** — server components cannot set `httpOnly` cookies. Routing
  single-org auto-select and org card "Select" links through `GET /api/org-select` allows the
  route handler to set the cookie before landing on `/assessments`.
- **Flat JSX over sub-components** — `OrgList`/`OrgCard` abstraction was premature at this
  scale. Deferred to #89 once repo count data is available.
- **Static link list over dropdown** — `OrgSwitcher` uses plain `<a>` links. No JS interaction
  needed at this stage. Full dropdown deferred.
- **Two-query pattern** — Supabase nested select (`organisations(*)`) produces `SelectQueryError`
  in TypeScript. Two typed queries + `flatMap` join is cleaner and avoids type assertions.
- **Cost script extraction** — shared `scripts/query-feature-cost.py` is the single source of
  truth for cost querying. Both `/feature` and `/feature-end` call it. Fixes the inflated-cost
  bug and adds label export in one place.
