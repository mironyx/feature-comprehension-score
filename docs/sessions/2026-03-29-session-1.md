### Bug fixes

Three bugs found and fixed during smoke testing:

**1. Missing `read:org` OAuth scope** (`src/app/auth/sign-in/SignInButton.tsx`)
- `GET /user/orgs` returned 403 — token lacked `read:org`.
- Fix: added `read:org` to `signInWithOAuth` scopes.

**2. Wrong `onConflict` target** (`src/lib/github/installation-handlers.ts`)
- `upsertRepos` used `onConflict: 'github_repo_id'` but the unique constraint is `(org_id, github_repo_id)`.
- Caused Postgres error `42P10` on every installation webhook.
- Fix: changed to `onConflict: 'org_id,github_repo_id'`.

**3. Personal account installations not surfaced in UI** (issue #119 — not yet fixed)
- App installed on personal account `leonids2005`; webhook upserted org + 6 repos successfully.
- `syncOrgMembership` only queries `/user/orgs`, which returns GitHub Organisations only — personal accounts never appear there.
- Result: `installedOrgs.length === 0` → `user_organisations` deleted → org-select shows nothing.
- Fix deferred: issue #119 created and added to board.
