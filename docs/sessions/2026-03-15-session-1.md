## Decisions Made

2. **OSS models for V2** — captured in requirements as a cost reduction consideration. `LLMClient` port already abstracts the provider, so adding alternatives is an adapter change.
3. **Agentic artefact retrieval for V2** — the `additional_context_suggestions` data will inform whether automatic retrieval justifies the additional LLM cost.
