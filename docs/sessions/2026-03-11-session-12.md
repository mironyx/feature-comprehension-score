### 2. CLAUDE.md — TDD Discipline section

Replaced the single TDD bullet in "How to Work" with a dedicated section enforcing strict Red-Green-Refactor:

- RED: Write failing test, run it, confirm it fails for the right reason.
- GREEN: Minimum code to pass. No more.
- REFACTOR: Clean up while green. Apply SOLID here.
- Rules: one test at a time, no batching, behaviour through public interfaces, BDD naming.

### 3. CLAUDE.md — Coding Principles section

Added six principles as a concise reference:

- SOLID (apply during refactoring, not upfront ceremony)
- Clean Architecture (`src/lib/engine/` is pure domain — no framework imports, no I/O)
- Dependency Inversion at boundaries (ports and adapters)
- Functions over classes
- Types as documentation (discriminated unions, branded types, Zod schemas)
- FIRST tests

### 5. ADR-0009: Test Diamond Strategy

Replaced the classic pyramid (70/20/10) with a module-specific diamond:

| Module | Unit | Integration | E2E |
|--------|------|-------------|-----|
| Engine (pure logic) | 70% | 20% | 10% |
| API routes + webhooks | 20% | 70% | 10% |
| UI pages | — | — | 100% |
| **Overall** | ~35% | ~45% | ~20% |

Rationale: integration tests give the best confidence-to-cost ratio for boundary-heavy code (Supabase RLS, GitHub webhooks). Engine stays unit-heavy because it's pure functions with no I/O. Coverage targets unchanged.

## Decisions

1. **Test diamond over pyramid** — Module-specific ratios matching the nature of each module type. Captured as ADR-0009.
2. **Tester + Developer combined** — One person wearing both hats is fine. The discipline is the Red-Green-Refactor sequence, not the role separation or commit boundary.
3. **CLAUDE.md as enforcement mechanism** — TDD rules and coding principles baked into CLAUDE.md so every Claude Code session inherits them automatically. Inspired by community patterns (alexop.dev TDD skill, thoughtbot TDD article).
4. **Separate test scripts** — `test:unit` vs `test:integration` allows fast feedback during TDD (run unit only) while keeping the full suite for CI.
