## Decisions made

- **Admin detection per-page, not in nav bar:** The LLD ┬¦2.6 described admin detection as a
  nav bar concern. In practice each page runs its own membership query ŌĆö the nav bar pattern
  adds complexity without benefit at this scale.
- **Shared `isOrgAdmin` helper:** Extracted to `src/lib/supabase/membership.ts` rather than
  inlining in both pages. Simple function, clear single responsibility.
- **Cumulative validation (post-PR discussion):** Changed from early-return `string | null`
  to `string[]` collecting all errors ŌĆö better UX for a 5-field form where multiple required
  fields can be empty simultaneously.
- **`React.SyntheticEvent` over `React.FormEvent`:** `React.FormEvent` is deprecated in
  React 19 type definitions. `SyntheticEvent<HTMLFormElement>` is the current replacement.
