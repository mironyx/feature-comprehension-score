## Decisions made

- **No spinner** — the LLD mentioned "muted / spinner" but only opacity muting (`opacity: 0.6`) was used. A static muted label avoids adding a client component and is sufficient for MVP.
- **`STATUS_LABELS` lookup table** — used a module-level `Record<string, string>` constant instead of a `switch` statement; easier to extend without touching conditionals.
- **Test file split** — component tests and page tests separated into two files (under a new `assessments/` subdirectory) rather than growing the existing `assessments.test.ts` further.
