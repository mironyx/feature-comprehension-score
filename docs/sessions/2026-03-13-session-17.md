## Decisions

- **Single `extractFromPRs(prNumbers: number[])` method** — not separate methods for single vs batch. Single PR = `[42]`, FCS batch = `[42, 43, 44]`.
- **Installation token for all artefact extraction** — both PRCC and FCS flows use the GitHub App installation token.
- **File listing + selective content** — `file_listing` for all changed files; `file_contents` only for top-N by lines changed.
- **Functions over class for prompt builder** — defer `PromptBuilder` class until scoring and relevance builders arrive.
- **GitHub Action approach** — documented as V2 option for security-conscious orgs.
